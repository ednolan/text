Function compare

boost::text::compare

Synopsis

// In header: <boost/text/collate.hpp>


int compare(text_sort_key const & lhs, text_sort_key const & rhs);

Description

Returns 0 if the given sort keys are equal, a value < 0 if lhs is less than rhs, and a value > 0 otherwise.
