Function template ends_encoded

boost::text::v1::ends_encoded

Synopsis

// In header: <boost/text/transcode_iterator.hpp>


template<utf8_iter I> 
  requires constexpr std::random_access_iterator< I > bool 
  ends_encoded(I first, I last);

Description

Returns true iff [first, last) is empty or the final UTF-8 code units in [first, last) form a valid Unicode code point.

Returns true iff [first, last) is empty or the final UTF-16 code units in [first, last) form a valid Unicode code point.
