Function template stream_safe_copy

boost::text::v1::stream_safe_copy

Synopsis

// In header: <boost/text/stream_safe.hpp>


template<typename CPIter, typename Sentinel, typename OutIter> 
  auto stream_safe_copy(CPIter first, Sentinel last, OutIter out);

Description

Writes sequence [first, last) to out, ensuring Stream-Safe Text Format.

This function only participates in overload resolution if CPIter models the CPIter concept.
