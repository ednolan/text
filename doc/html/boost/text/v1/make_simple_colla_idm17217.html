Function template make_simple_collation_searcher

boost::text::v1::make_simple_collation_searcher

Synopsis

// In header: <boost/text/collation_search.hpp>


template<typename GraphemeRange, typename BreakFunc> 
  unspecified make_simple_collation_searcher(GraphemeRange && r, 
                                             BreakFunc break_fn, 
                                             collation_table const & table, 
                                             collation_flags flags = collation_flags::none);

Description

Returns a simple_collation_searcher that will find the pattern r. Any occurence of the pattern must be found starting at and ending at a boundary found by break_fn (e.g. a grapheme or word boundary).

This function only participates in overload resolution if GraphemeRange models the GraphemeRange concept.

BreakFunc must be an invocable type whose signature is CPIter (CPIter first, CPIter it, Sentinel last), where CPIter is decltype(r.begin().base()) and Sentinel is decltype(r.end().base()).
