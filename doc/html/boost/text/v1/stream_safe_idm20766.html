Function template stream_safe

boost::text::v1::stream_safe

Synopsis

// In header: <boost/text/stream_safe.hpp>


template<typename CPRange> auto stream_safe(CPRange && r);

Description

Copies the stream-safe portion of r to the beginning of r, and returns an iterator to the end of the copied range.

This function only participates in overload resolution if CPRange models the CPRange concept.

See Also:

https://unicode.org/reports/tr15/#Stream_Safe_Text_Format
