Function template transcode_to_utf8

boost::text::v1::transcode_to_utf8

Synopsis

// In header: <boost/text/transcode_algorithm.hpp>


template<typename Ptr, std::output_iterator< uint8_t > O> 
  transcode_result< Ptr, O > transcode_to_utf8(Ptr p, O out);

Description

Copies the code points in the range [p, null_sentinel) to out, changing the encoding to UTF-8.
