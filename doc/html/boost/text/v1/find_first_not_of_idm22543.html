Function template find_first_not_of

boost::text::v1::find_first_not_of

Synopsis

// In header: <boost/text/string_algorithm.hpp>


template<utf_range_like R1, utf_range_like R2> 
  unspecified find_first_not_of(R1 && r1, R2 && r2);

Description

A Unicode-friendly version of std::string::find_first_not_of(). Compatibility overloads exist to support operating on code points and graphemes; ranges and null-terminated strings; and different UTF formats. This means that this overload set is compatible with text, rope, etc. When R1 is not a pointer, the result is returned as a borrowed_subrange_t in C++20 and later.
