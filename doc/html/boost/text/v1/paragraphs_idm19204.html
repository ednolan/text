Function template paragraphs

boost::text::v1::paragraphs

Synopsis

// In header: <boost/text/paragraph_break.hpp>


template<typename CPRange> unspecified paragraphs(CPRange && range);

Description

Returns a lazy range of the code point ranges delimiting paragraphs in range.

This function only participates in overload resolution if CPRange models the CPRange concept.
