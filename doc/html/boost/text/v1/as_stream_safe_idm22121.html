Function template as_stream_safe

boost::text::v1::as_stream_safe

Synopsis

// In header: <boost/text/stream_safe.hpp>


template<code_point_range R> unspecified as_stream_safe(R && r);

Description

Returns a stream_safe_view of the range r. The result is returned as a borrowed_view_t in C++20 and later.

See Also:

https://unicode.org/reports/tr15/#Stream_Safe_Text_Format
