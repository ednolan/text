Global requires

boost::text::v1::requires

Synopsis

// In header: <boost/text/transcode_iterator.hpp>

requires requires;

Description

Returns a inserting iterator that transcodes from UTF-8 to UTF-8, UTF-16, or UTF-32. Which UTF the iterator transcodes to depends on sizeof(Cont::value_type): 1 implies UTF-8; 2 implies UTF-16; and any other size implies UTF-32.

Returns a inserting iterator that transcodes from UTF-16 to UTF-8, UTF-16, or UTF-32. Which UTF the iterator transcodes to depends on sizeof(Cont::value_type): 1 implies UTF-8; 2 implies UTF-16; and any other size implies UTF-32.

Returns a inserting iterator that transcodes from UTF-32 to UTF-8, UTF-16, or UTF-32. Which UTF the iterator transcodes to depends on sizeof(Cont::value_type): 1 implies UTF-8; 2 implies UTF-16; and any other size implies UTF-32.

Returns a back-inserting iterator that transcodes from UTF-8 to UTF-8, UTF-16, or UTF-32. Which UTF the iterator transcodes to depends on sizeof(Cont::value_type): 1 implies UTF-8; 2 implies UTF-16; and any other size implies UTF-32.

Returns a back-inserting iterator that transcodes from UTF-16 to UTF-8, UTF-16, or UTF-32. Which UTF the iterator transcodes to depends on sizeof(Cont::value_type): 1 implies UTF-8; 2 implies UTF-16; and any other size implies UTF-32.

Returns a back-inserting iterator that transcodes from UTF-32 to UTF-8, UTF-16, or UTF-32. Which UTF the iterator transcodes to depends on sizeof(Cont::value_type): 1 implies UTF-8; 2 implies UTF-16; and any other size implies UTF-32.

Returns a front-inserting iterator that transcodes from UTF-8 to UTF-8, UTF-16, or UTF-32. Which UTF the iterator transcodes to depends on sizeof(Cont::value_type): 1 implies UTF-8; 2 implies UTF-16; and any other size implies UTF-32.

Returns a front-inserting iterator that transcodes from UTF-16 to UTF-8, UTF-16, or UTF-32. Which UTF the iterator transcodes to depends on sizeof(Cont::value_type): 1 implies UTF-8; 2 implies UTF-16; and any other size implies UTF-32.

Returns a front-inserting iterator that transcodes from UTF-32 to UTF-8, UTF-16, or UTF-32. Which UTF the iterator transcodes to depends on sizeof(Cont::value_type): 1 implies UTF-8; 2 implies UTF-16; and any other size implies UTF-32.
