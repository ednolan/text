Function template to_lower

boost::text::v1::to_lower

Synopsis

// In header: <boost/text/case_mapping.hpp>


template<code_point_range R, std::weakly_incrementable O> 
  requires std::indirectly_copyable< std::ranges::iterator_t< R >, O > case_mapping_result< std::ranges::borrowed_iterator_t< R >, O > 
  to_lower(R && r, O out, case_language lang = case_language::other);

Description

Writes the code point sequence comprising the lower-case form of r to out, using language-specific handling as indicated by lang. The normalization of the result is undefined.
